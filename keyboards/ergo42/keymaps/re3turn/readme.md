# The Default Helix Layout

## Layout

### BASE(QWERTY)

```text
 ,------------------------------------------------.   ,------------------------------------------------.
 |   `  |   Q  |   W  |   E  |   R  |   T  |  [   |   |  ]   |   Y  |   U  |   I  |   O  |   P  |  =   |
 |------+------+------+------+------+------+------|   |------+\-----+------+------+------+------+------|
 | Alt  |   A  |   S  |   D  |   F  |   G  |  (   |   |  )   |   H  |   J  |   K  |   L  |   ;  |  '   |
 |------+------+------+------+------+------+------|   |------+------+------+------+------+------+------|
 | Shift|   Z  |   X  |   C  |   V  |   B  |  {   |   |  }   |   N  |   M  |   ,  |   .  |   /  |  -   |
 |------+------+------+------+------+------+------|   |------+------+------+------+------+------+------|
 | Ctrl | GUI  | Alt  | IME  | Esc/ |Space/| Tab/ |   | BS/  |Enter/| Del  | GUI+ | Alt  | GUI  |  \   |
 |      |      |      |      | Ctrl |~SYMB | Shift|   | Alt  |~META |      | Tab  |      |      |      |
 `------------------------------------------------'   `------------------------------------------------'
```

## Compile

go to qmk top directory.

```bash
cd qmk_firmware
```

build

```bash
make ergo42/rev1:re3turn
```

flash to keyboard

```bash
make ergo42/rev1:re3turn:avrdude
```
